# Save Learning

Review everything we did in this session and identify anything worth preserving 
for future sessions. Look for:

- **Reusable instruction sets** → save as a skill in `personal-ai/skills/`
- **Repo-specific workflows** → save as a command in `.claude/commands/`
- **Reusable prompts** → save as a prompt in `personal-ai/prompts/`
- **Project conventions** → add to the relevant `CLAUDE.md`
- **Global preferences** → add to `personal-ai/docs/custom-instructions/claude-global.md`

For each candidate:
1. Name it clearly (e.g. `skills/things-task-creation.md`)
2. Draft the content
3. Show it to Andy and ask for confirmation before saving
4. After confirmation, create the file and commit it with a conventional commit message

Do not save anything generic or obvious. Only save things that would meaningfully 
change the quality of a future session if Claude didn't have them.

After saving, summarize what was captured and where it lives.
